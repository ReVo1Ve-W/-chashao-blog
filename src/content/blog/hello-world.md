这是我博客的第一篇正式文章，主要是测试一下写作流程：在 VS Code 里新建一个 `.md` 文件，写点内容，保存后就能在本地预览看到效果。

## 这个博客是怎么搭的

用的是 Astro 框架，纯 Markdown 写作，不需要数据库和后台管理系统。以后写文章的流程大概是：

1. 在 `src/content/blog/` 目录下新建一个 `.md` 文件
2. 写好 frontmatter（标题、描述、发布日期），然后正文随便写
3. 保存，本地跑一下预览确认没问题
4. 提交到 Git，推送后自动部署上线
